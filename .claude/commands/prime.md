# /prime — Session Start

Bitte führe die folgenden Schritte durch, um unsere Session zu starten:

1. Lies alle Dateien im `context/`-Ordner:
   - `context/business-info.md`
   - `context/personal-info.md`
   - `context/strategy.md`
   - `context/current-data.md`

2. Prüfe, ob es Dateien im `context/import/`-Ordner gibt, und lies diese ebenfalls ein.

3. Gib mir eine kurze Zusammenfassung (3-5 Punkte) von dem, was du über mein Business weißt.

4. Weise mich auf fehlende oder veraltete Informationen hin.

5. Frage mich, woran ich heute arbeiten möchte.

Starte mit: "Session gestartet! Ich habe deinen Kontext geladen. Hier ist was ich weiß: ..."
