# /create-plan — Implementierungsplan erstellen

Erstelle einen detaillierten, umsetzbaren Plan für eine neue Aufgabe oder ein neues Projekt.

## Ablauf

1. Stelle mir diese Fragen (eine nach der anderen):
   - "Was möchtest du umsetzen? Beschreibe das Ziel."
   - "Warum ist das wichtig für dein Business?"
   - "Gibt es eine Deadline oder Zeitvorstellung?"
   - "Was hast du bereits dazu gemacht oder recherchiert?"
   - "Welche Ressourcen stehen dir zur Verfügung? (Zeit, Budget, Tools)"

2. Analysiere die Antworten im Kontext meines Businesses (nutze `context/`-Dateien).

3. Erstelle einen strukturierten Plan mit:
   - **Ziel** — Was wird am Ende erreicht sein?
   - **Voraussetzungen** — Was muss zuerst vorhanden sein?
   - **Phasen** — Grobe Unterteilung in 2-4 Phasen
   - **Konkrete Aufgaben** — Einzelne, abhakbare To-dos pro Phase
   - **Zeitschätzung** — Realistische Zeit pro Aufgabe
   - **Potenzielle Hindernisse** — Was könnte schiefgehen?
   - **Erfolgsmessung** — Woran erkenne ich, dass es funktioniert hat?

4. Speichere den Plan als Datei in `plans/plan-[datum]-[thema].md`.

5. Frage: "Soll ich direkt mit der Umsetzung beginnen? Dann nutze /implement."

Starte mit: "Ich erstelle einen Plan für dich. Lass uns mit ein paar Fragen beginnen..."
