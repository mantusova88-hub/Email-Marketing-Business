# /capture — Ideen & Aufgaben schnell festhalten

Schnell-Capture für Ideen, To-dos, Erkenntnisse — ohne Unterbrechung des Flows.

## Ablauf

1. Frage sofort (keine langen Einleitungen):
   "Was möchtest du festhalten? (Idee / Aufgabe / Erkenntnis / Content-Idee)"

2. Nimm die Eingabe entgegen — auch Sprachnotiz-Text, rough & unfertig ist okay

3. Kategorisiere automatisch:
   - Idee → `outputs/ideen.md`
   - Aufgabe → `outputs/aufgaben.md`
   - Content-Idee → `outputs/content-ideen.md`
   - Erkenntnis / Lernen → `outputs/erkenntnisse.md`

4. Speichere mit Datum + kurzer Zusammenfassung

5. Frage: "Noch etwas? Oder weiter mit deiner Aufgabe?"

## Format beim Speichern

```
## [Datum] — [Kategorie]
[Inhalt genau wie eingegeben]
→ Nächster Schritt: [falls eindeutig]
```

Starte mit: "Schieß los — was willst du festhalten?"
