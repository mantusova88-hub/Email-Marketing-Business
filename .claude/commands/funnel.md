# /funnel — Funnel-Architektur für Simones Business

Plane und beschreibe einen kompletten Marketing-Funnel: von der ersten Berührung bis zum Kauf — Evergreen oder Launch.

## Funnel-Typen

- **Lead-Magnet-Funnel** — Freebie → E-Mail-Liste → Welcome-Sequenz → Angebot
- **Webinar-Funnel** — Anmeldung → Erinnerungen → Webinar → Sales-Sequenz
- **Evergreen-Funnel** — läuft automatisch, kein Launch nötig
- **Value-Ladder** — Einstieg günstig → Upsell → High-Ticket

## Ablauf

1. Frage: "Welcher Funnel-Typ? Oder: Was willst du erreichen?"
2. Frage: "Was ist das Hauptangebot am Ende des Funnels?"
3. Frage: "Was ist der Einstiegspunkt? (Freebie / Webinar / Quiz / Challenge)"
4. Frage: "Welche Tools nutzt du? (E-Mail-Provider, Landingpage-Tool)"

5. Erstelle die vollständige Funnel-Map:

---

**FUNNEL-ÜBERSICHT**

```
[Traffic-Quelle] → [Opt-in] → [Bestätigung] → [Willkommensseite]
      ↓
[Welcome-Sequenz: X E-Mails]
      ↓
[Angebot: Name / Preis]
      ↓
[Upsell / Downsell]
```

**Schritt-für-Schritt-Beschreibung:**
Für jeden Funnel-Schritt: Was passiert? Welcher Text/Content? Welches Tool?

**E-Mail-Sequenz-Plan:**
Alle E-Mails mit Betreffzeile, Timing und Ziel

**Technischer Setup-Plan:**
Welches Tool für welchen Schritt + Verbindungen

---

6. Ergänze: Optimierungs-Tipps und häufige Fehler vermeiden

7. Speichere unter `plans/funnel-[datum]-[name].md`

Starte mit: "Funnel-Planung! Was ist dein Ziel und dein Hauptangebot?"
