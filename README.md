# Email Marketing Business

E-Mail Marketing Business für selbständige Mamas

## Ordnerstruktur

| Ordner | Inhalt |
|---|---|
| `01_Kunden/` | Kundenprojekte, Verträge, Berichte je Kunde |
| `02_Kampagnen/` | Vorlagen, aktive und archivierte Kampagnen |
| `03_Inhalte/` | Texte, Bilder, Banner, Logos |
| `04_Listen/` | E-Mail-Listen, Segmente, Blacklists |
| `05_Analyse/` | Berichte, KPIs, A/B-Tests |
| `06_Automatisierungen/` | Workflows, Trigger, E-Mail-Sequenzen |
| `07_Finanzen/` | Rechnungen, Angebote, Finanzübersichten |
| `08_Rechtliches/` | DSGVO, Verträge, Einwilligungen |
| `09_Tools_Zugaenge/` | API-Keys, Tool-Dokumentation |
| `10_Ressourcen/` | Schulungen, Best Practices, Brancheninfos |

## Wichtige Hinweise

- **DSGVO**: Alle E-Mail-Listen nur mit nachweisbarer Einwilligung speichern (`08_Rechtliches/Einwilligungen/`)
- **Kunden**: Pro Kunde einen eigenen Unterordner unter `01_Kunden/`
- **Kampagnen**: Abgeschlossene Kampagnen in `02_Kampagnen/Archiv/` verschieben
- **API-Keys**: Niemals im Klartext in Code-Dateien speichern (`09_Tools_Zugaenge/API_Keys/`)
